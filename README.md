# 🍀 다시마켓 (Dasimarket)
**통합형 중고 & 자체 굿즈 거래 및 커뮤니티 플랫폼**

---

## 📌 프로젝트 개요 (Project Overview)

다시마켓은 단순한 중고 거래를 넘어, 자체 제작된 굿즈를 판매하고 사용자 간의 활발한 소통을 지원하는 통합형 마켓플레이스입니다.  
안전한 거래 환경과 사용자 친화적인 유저 커뮤니티 공간을 제공하여, 사용자들에게 새로운 가치 소비 경험을 제공합니다.

---

## 🛠️ 개발 환경 및 기술 스택 (Tech Stack)

본 프로젝트는 안정적인 백엔드 환경과 사용자 친화적인 프론트엔드를 구축하기 위해 아래 기술 스택을 활용했습니다.

| 구분 (Category)       | 기술 (Technology)       | 설명 (Role) |
|-----------------------|-----------------------|------------|
| 핵심 백엔드           | JAVA, Spring Boot, Gradle | 핵심 비즈니스 로직 및 서버 구동 환경 |
| 템플릿/프론트엔드      | HTML5, CSS3, JavaScript, Thymeleaf | 웹 페이지 구조 및 서버 사이드 렌더링 |
| 데이터베이스           | ORACLE DB             | 영구 데이터 저장 및 관리 |
| 결제/외부 API          | PortOne, KAKAO API    | 안전한 금융 거래 처리 및 소셜 로그인 |

---

## 🌟 프로젝트 주요 서비스 기능 (Key Services)

### 1️⃣ 중고 상품 거래
- **상품 등록 및 관리**  
  중고 상품의 상세 정보, 상태, 카테고리, 가격 등을 등록 및 수정 가능
- **유저 간 채팅**  
  구매자와 판매자가 직접 소통하며 직거래 가능

### 2️⃣ 자체 굿즈 판매
- **굿즈 전용 상점**  
  프로젝트 자체 제작 상품(예: 캐릭터 쿠션 등)을 판매하는 독립적인 쇼핑 공간 제공
- **상품 관리 기능**  
  상품 등록, 수정, 재고 관리 등 운영자 중심 기능 지원

### 3️⃣ 유저 커뮤니티
- **자유 게시판**  
  모든 사용자가 자유롭게 소통하고 정보를 교환할 수 있는 게시판 제공
- **활동 참여 독려**  
  게시글 작성, 댓글, 추천 기능 등을 통해 사용자 활동 참여 촉진
